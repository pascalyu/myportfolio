Todogether

TodoList mais pour un groupe

Développé en 3 jours, je l'ai crée surtout pour découvrir le bundle softdelete.
Il est très simple, il y a que 3 tables: User, Task,Team
Fonctionnalités
  • Inscription/Login
  • Créer un groupe
  • Rejoindre un groupe avec un code
  • Affichage des tâches
  • Créer une tâche
  • Faire une tâche
  • Supprimer une tâche
  • Undo une tâche
Technologies utilisés
  • Symfony 5
  • MySQL
  • Bootstrap
  • GIT
